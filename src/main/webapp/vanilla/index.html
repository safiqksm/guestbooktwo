All my problems went away with the newer version of TomEE whitch detects and uses jackson for json serialization. Just add this to pom.xml:

        
<dependency>
    <groupId>com.fasterxml.jackson.jaxrs</groupId>
    <artifactId>jackson-jaxrs-json-provider</artifactId>
    <version>2.3.2</version>
</dependency>
